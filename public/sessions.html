Session ID Status API Key Actions

API Documentation

Send a Message

Endpoint: POST /message/:sessionId

Headers:

{
    "x-api-key": "your_api_key"
}

Body:

{
    "id": "recipient_id",
    "text": "message_text"
}

Set Webhook URL

Endpoint: POST /set-webhook/:sessionId

Headers:

{
    "x-api-key": "your_api_key"
}

Body:

{
    "webhookUrl": "webhook_url"
}

Generate New API Key

Endpoint: POST /genapi/:sessionId

Headers:

{
    "x-api-key": "your_api_key"
}

Get QR Code

Endpoint: GET /qr/:sessionId

Headers:

{
    "x-api-key": "your_api_key"
}
