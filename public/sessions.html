API Documentation

Send a Message

Endpoint: POST /message/:sessionId

Headers:

{
    "x-api-key": "your_api_key"
}

Body:

{
    "id": "recipient_id",
    "text": "message_text"
}

Check if a Number Exists on WhatsApp

Endpoint: GET /checkno/:sessionId/:phone

Headers:

{
    "x-api-key": "your_api_key"
}

Send an Image

Endpoint: POST /sendimage/:sessionId

Headers:

{
    "x-api-key": "your_api_key"
}

Body:

{
    "id": "recipient_id",
    "url": "image_url",
    "caption": "optional_caption"
}
